## Лабораторная работа №4. Симуляция с пользовательсĸими ĸоллеĸциями и псевдослучайной моделью.
## Цель:
Создать предметную модель "Библиотека", включающую классы Book, BookCollection и Library. А также реализовать две пользовательские коллекции (BookCollection и IndexDict), осуществляющие протоколы последовательности и отображения (магические методы __iter__, __len__, __getitem__ и поддержку операций добавления и удаления)

## События программы:
1) Добавить книгу
2) Удалить книгу
3) Найти книгу по автору
4) Найти книгу по году
5) Найти книгу по жанру
6) Взять книгу
7) Обновить индекс
8) Вернуть книгу

## Описание:
Выполняет симуляцию в steps шагов, использует random, при заданном seed генерирует идентичную последовательность событий, на ĸаждом шаге симуляции происходит одно случайное событие, выбранное из списĸа, все события симуляции выводиться в ĸонсоль в формате теĸстового лога.

## Как работает:
1) Симуляция говорит: "Нужно добавить книгу"
2) Вызывается generate_book()
3) Функция берет случайные данные из BOOK_INFORMATION
4) Создает объект класса Book с этими данными
5) Возвращает  готовую книгу с симуляцией
6) Симуляция добавляет книгу в библиотеку
